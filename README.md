
# 💸 Expense Manager (SwiftUI)

A modern and minimal Expense Manager iOS app built using **SwiftUI**, focused on clean UI, performance, and real-time financial insights.

---
## 🚀 Features

- 📊 Dashboard with balance & analytics
- ➕ Add Expense with category selection
- 📅 Transaction history with filters
- 📈 Charts (Monthly, Category breakdown)
- 🌙 Dark & Light Mode support
- 💾 Local storage (CoreData/UserDefaults)
- ⚡ Smooth animations & SwiftUI transitions

---

## 🧠 Tech Stack

- Swift 5
- SwiftUI
- MVVM Architecture
- Combine (optional)
- Charts Framework
- CoreData (for persistence)
